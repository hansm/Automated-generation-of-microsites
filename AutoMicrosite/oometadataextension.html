OpenAjax Metadata 1.0 extension for Automated generation of microsites

This extension to OpenAjax Metadata adds some necessary attributes for automated generation of microsites.

<widget>

New attributes introduced to <widget> element:

'max-height'

Maximum widget width in pixels that the developer tool may resize the widget to. Developer tool may not resize the height of the widget without 'min-height' and 'max-height' attributes. Optional.

'min-height'

Minimum widget height in pixels that the developer tool may resize the widget to. Developer tool may not resize the height of the widget without 'min-height' and 'max-height' attributes. Optional.

'max-width'

Maximum widget width in pixels that the developer tool may resize the widget to. Developer tool may not resize the width of the widget without 'min-width' and 'max-width' attributes. Optional.

'min-width'

Minimum widget width in pixels that the developer tool may resize the widget to. Developer tool may not resize the width of the widget without 'min-width' and 'max-width' attributes. Optional.

<category>

'iri'

Schema.org ontology category.

<content>

'iri'

URL of regular OpenAjax widget.
